# 🧠 Math Quiz Game in C++

A simple console-based quiz game built using C++ during my learning journey.

---

## 🎯 Project Idea
This project generates random math questions for the user and evaluates their answers based on different difficulty levels and operation types.

---

## ⚙️ Features
- Random number generation
- Multiple difficulty levels:
  - Easy
  - Medium
  - Hard
  - Mix
- Multiple operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division
  - Mix operations
- Score tracking:
  - Right answers
  - Wrong answers
- Final result (Pass / Fail)
- Colored output for better user experience

---

## 🧩 Concepts Used
- Functions
- Structs
- Enums
- Random numbers
- Input validation
- Loops
- Basic game logic

---

## 📊 Game Flow
1. User selects number of questions
2. Selects difficulty level
3. Selects operation type
4. Answers generated questions
5. Final score is displayed

---

## 🚀 How to Run
1. Open project in Visual Studio or any C++ IDE
2. Build and run the program
3. Follow on-screen instructions
